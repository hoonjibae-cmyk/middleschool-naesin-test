# 중등 내신 모의고사 제조기 v14

## v14 핵심 수정

이번 버전은 v10~v13에서 반복된 오류를 전반적으로 정리한 안정화 버전입니다.

1. 30문항 생성은 한 번에 처리하지 않고 4문항 단위로 분할 생성합니다.
2. Claude 응답 JSON 파싱을 더 안정적으로 처리하도록 서버 파서를 교체했습니다.
3. Claude 응답이 JSON 배열 앞의 여는 대괄호를 누락하는 경우를 보완했습니다.
4. API 요청 프롬프트를 서버에서 다시 압축해 타임아웃과 JSON 오류 가능성을 줄였습니다.
5. 문항 수를 +/− 버튼뿐 아니라 숫자 직접 입력으로도 조정할 수 있게 했습니다.
6. 총 문항, 객관식, 서술형 입력값이 서로 맞지 않을 때 자동 보정합니다.

## 권장 Vercel 환경변수

ANTHROPIC_API_KEY=원장님 API Key
ANTHROPIC_FAST_MODEL=claude-haiku-4-5
ANTHROPIC_MAX_TOKENS=1600

기존에 ANTHROPIC_FAST_MODEL 값이 claude-3-5-haiku-latest로 되어 있으면 반드시 변경해야 합니다.

1. 8문항 테스트
2. 12문항 테스트
3. 20문항 테스트
4. 30문항 테스트

30문항은 내부적으로 4문항씩 8개 구간으로 나누어 생성됩니다.
