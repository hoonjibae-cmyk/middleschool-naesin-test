# 중등 내신 모의고사 제조기 — 미강중3 기출 스타일 강화 버전

## 이번 버전의 핵심 변경

- 미강중3 2025년 2학기 2차 기말고사 프로필을 실제 내신 스타일 중심으로 강화했습니다.
- 기본 문항 수를 8문항, 객관식 8문항, 서술형 0문항으로 조정했습니다.
- 같은 지문을 여러 번 반복 출력하지 않도록 프롬프트와 화면 렌더링을 보완했습니다.
- 기출 프로필에서 서술형이 0문항인 경우, 서술형 답란만 비어 있는 문제를 생성하지 않도록 지시를 강화했습니다.
- 미강중3 스타일에 맞춰 내용 불일치, 제목/요지, 빈칸, 어법, 대화문 Q&A, 모두 고르기/개수 세기/as 용법 구분형이 우선 반영되도록 했습니다.

## 배포 방법

GitHub 저장소의 기존 파일을 이 ZIP 안의 파일로 전체 교체한 뒤 Commit 하세요.
Vercel이 GitHub와 연결되어 있다면 자동으로 재배포됩니다.

필요한 환경변수:

- ANTHROPIC_API_KEY
- ANTHROPIC_MODEL 선택 사항
- ANTHROPIC_MAX_TOKENS 선택 사항

## 권장 테스트

1. 학교: 미사강변중학교
2. 학년: 3학년
3. 기출: 2025년 2학기
4. 문항: 기본값 8문항 그대로 테스트

생성 결과에서 확인할 것:

- 지문이 반복 인쇄되지 않는지
- 서술형 0문항인데 서술형 답란이 생기지 않는지
- 오답 선지가 지문 근거를 살짝 바꾼 함정형인지
- 어법/어휘 문항이 지문 속 표현 기반인지
