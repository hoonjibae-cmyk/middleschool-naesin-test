# 중등 내신 모의고사 제조기 — 30문항 분할 생성 버전

## 핵심 변경

이번 버전은 30문항 실사용을 위해 한 번에 전체 문항을 생성하지 않습니다.
브라우저가 내부적으로 6문항 단위로 여러 번 `/api/generate-exam`을 호출하고, 마지막에 전체 문항을 병합합니다.

예시: 30문항 생성 시

1. 1~6번 생성
2. 7~12번 생성
3. 13~18번 생성
4. 19~24번 생성
5. 25~30번 생성
6. 전체 병합 및 PDF 렌더링

## 장점

- Vercel 60초 타임아웃 가능성 감소
- Claude JSON/도구 출력 실패 가능성 감소
- 각 구간별 문제 품질 유지
- 30문항 이상 확장 가능성 확보

## 권장 환경변수

- `ANTHROPIC_API_KEY`: 필수
- `ANTHROPIC_FAST_MODEL`: 권장, 예: `claude-3-5-haiku-latest`
- `ANTHROPIC_MAX_TOKENS`: 권장, 예: `1900`

## 사용상 주의

30문항은 내부적으로 5회 생성되므로 8문항보다 시간이 오래 걸립니다. 정상입니다.
생성 버튼에는 현재 생성 중인 번호 구간이 표시됩니다.
