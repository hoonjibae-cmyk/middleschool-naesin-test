# 목동유쌤 중등 내신 모의고사 생성기 v17

## 핵심 변경

v17은 기출 스타일만으로 Claude가 새 지문을 창작하던 구조에서 벗어나, 저장된 교과서 유닛 자료를 문제의 실제 소스로 사용하는 첫 버전입니다.

추가된 파일:

```txt
/data/textbook-units.json
```

현재 샘플 등록 자료:

```txt
미사강변중학교 / 3학년 / 2026년 1학기
동아 영어 3 (윤정미)
Lesson 8 The Joseon Dynasty Through Paintings
Reading: Munjado, a Window into the Joseon Dynasty
```

## 작동 방식

- 화면의 `저장 교과서 자료` 영역에서 등록된 본문/대화문을 선택합니다.
- 주교재 비율이 100%이고 외부지문이 0%일 때는 선택된 저장 교과서 자료만 기반으로 출제하도록 프롬프트가 구성됩니다.
- 외부지문 비율을 올렸을 때만 저장 자료 밖의 외부 소재 사용을 허용하는 방향으로 설계했습니다.

## 주의

이 버전은 관리자 PDF 업로드 자동 추출 기능까지 포함한 버전은 아닙니다. 업로드된 Lesson 8 PDF를 기준으로 핵심 본문/대화문을 수동 구조화한 `textbook-units.json` 샘플을 포함합니다.

교과서/출판사 자료는 저작권 이슈가 있으므로, 실제 운영 시에는 공개 GitHub 저장소가 아니라 서버 전용 비공개 저장소에 보관하는 구조로 이전해야 합니다.


## v19 레이아웃 수정

- 실제 문항 높이를 측정해 A4 한 페이지의 왼쪽 열을 위에서 아래로 먼저 채우고, 그 다음 오른쪽 열을 채우는 방식으로 변경했습니다.

## v22 추가 수정
- 선택지 밑줄을 일괄 삭제하는 방식에서 조건부 허용 방식으로 변경했습니다.
- 발문이 밑줄 친 선택지/표현을 직접 비교하도록 요구하고, 복수 선택지에 의도적으로 밑줄이 들어간 경우에는 밑줄을 표시합니다.
- 한 선택지에만 우연히 들어간 [u] 또는 __ 표시처럼 오류 가능성이 높은 밑줄은 자동 제거합니다.

## v23 추가 수정

- 선택지 앞에 붙은 불필요한 ⓐ~ⓔ 표식은 자동 제거합니다.
- 선택지 밑줄은 v22의 조건부 허용 방식을 유지합니다.

## v24 변경 사항
- v23의 어법 후보 중복 선택지 제거와 선택지 ⓐ~ⓔ 정리 로직을 유지했습니다.
- 30문항 생성 시 전체 Blueprint를 고정하여 대화문, 문자도 정의, 효 이야기, 강/잉어, 대나무/연꽃, 교육적 의미, 종합 고난도 순서로 범위를 분산하도록 했습니다.
- 내용일치/불일치 반복을 줄이고 지칭어, 빈칸, 문장삽입/순서, 어휘 의미, 어법, 요약문 완성 유형을 섞도록 프롬프트를 강화했습니다.
- 저장 교과서 자료가 선택된 경우 기출 프로필은 스타일에만 사용하고, 지문 내용은 저장 교과서 소스에서만 가져오도록 강화했습니다.
- PDF pagebreak 설정에서 q-page avoid 충돌을 제거하여 30문항 출력 시 빈 페이지가 생기는 현상을 줄였습니다.
